# Maldev-kata

My personal journey learning Windows malware development and offensive security. Each folder contains a self-contained implementation of a technique in C/C++.

---

## Topics (Currently)

- **1. Classic Injection** — different ways to execute code inside another process
- **2. Enumeration** — various methods to discover processes and modules
- **3. Hooking** — intercepting function calls at the userland level

---

## Building

Each folder has a `compile.bat`. Requires MSVC and Windows SDK (x64).
